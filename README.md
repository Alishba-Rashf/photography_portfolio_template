Photography Portfolio Template:
A modern, responsive, and minimal web template tailored for professional photographers to showcase their visual artistry. Built with clean, semantic web standards to ensure fast loading times and seamless cross-device compatibility.

# Key Features

* Responsive Grid: Automatically adapts to desktop, tablet, and mobile screens.
* Lightweight & Fast: Built entirely on vanilla technologies with zero heavy framework overhead.
* Clean Code Architecture:Well-structured CSS and modular JavaScript make customization effortless.

# Built With

* HTML5 – Semantic markup for enhanced accessibility and SEO.
* CSS3 – Flexbox, Grid layouts, and smooth transition animations.
* JavaScript (ES6+) – Interactive gallery features and dynamic UI components.

# Repository Structure


├── assets/             # Photography media assets (Images/Icons)

├── portfolioJs/        # Modular JavaScript functionality

├── index.html          # Main landing page & application entry point
